What is this all about?

This is a site created using the bare bones of HTML5.

It serves two purposes: 1. to test HTML5 functionality. 2. To create as simple as possible design and process to allow for frequent updates on the fly.

Twitter bootstrap is used to respond to multiple consuming devices

For now I keep the defaults CSS as much as possible

Codex will document my learning journey

Notes page is for testing text update, blog type thing

Pictures is for ... pictures

Films page is for testing film embeds and what else can be done with HTML5

References page is to present links to other people's stuff that I like

Lab is for the latest experiments
